Oracle Mesh#

A production-grade Kubernetes platform running on Oracle Cloud’s free tier ARM instance.

What It Is#

  • KIND Kubernetes cluster with 21 pods
  • Linkerd service mesh with automatic mTLS
  • Traefik ingress with path-based routing
  • HashiCorp Vault for secrets management
  • Langfuse for AI observability
  • PostgreSQL, Redis, MinIO, ClickHouse data layer

Cost#

$0/month — Everything runs on Oracle’s Always Free tier.
